# 💹 FinCalc Pro — Financial Calculator Tool

🌐 Live Website: https://maryanoor66-jpg.github.io/financial-calculator

---

## 📌 About This Project

FinCalc Pro is a free professional financial calculator tool built for financial analysts, students, and business professionals. No sign-up, no cost, instant results.

---

## 🛠️ Technologies Used

- HTML5 — Structure and layout of the website
- CSS3 — Styling, animations, and responsive design
- JavaScript — All calculator logic and financial formulas
- Google Fonts — Typography
- GitHub Pages — Free hosting and deployment
- Google Search Console — SEO and search indexing

---

## 📊 What This Tool Does

### 1. ROI Calculator
- Net Gain / Loss
- Total ROI percentage
- Annual ROI
- Payback period in years

### 2. Loan & EMI Calculator
- Monthly EMI calculation
- Total payment and total interest
- Full year-by-year amortisation schedule
- Supports EUR, USD, GBP, PKR

### 3. Profit & Loss Calculator
- Gross Profit and Gross Margin
- EBIT calculation
- Net Profit after tax
- Full P&L breakdown

---

## 🚀 How I Built & Deployed It

1. Planned the project and identified what financial analysts need most
2. Designed the UI with a professional navy and gold theme
3. Wrote all HTML, CSS, and JavaScript code from scratch
4. Created a GitHub repository and uploaded the project
5. Deployed live using GitHub Pages at zero cost
6. Verified with Google Search Console using HTML meta tag
7. Submitted for Google indexing

Total cost: zero. Total tools: a browser and GitHub account.
